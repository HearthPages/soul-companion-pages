## Implementation Steps (For You)

### Phase 1: Generate QR Code (5 minutes)

1. Go to **qr-code-generator.com**
2. Paste URL: `https://hearthandpages.netlify.app/?journal=soul-quiet`
3. Generate QR code
4. Download as PNG: `soul-quiet-qr.png`
5. Test it with your phone (scan to verify it loads)

### Phase 2: Add to The Quiet No PDF (10 minutes)

**Option A: If you have the Word source file:**
1. Open `The_Quiet_No_People_Pleasing.docx` (or source file)
2. Go to "How to Use This Workbook" page
3. Copy this text at the bottom:

```
Your Soul Companion

This workbook comes with a free AI Soul Companion — for the moments 
when you want to explore what's underneath your people-pleasing patterns, 
or when you're learning to hear your own voice again.

Open your workbook to the section you're on. Scan the QR code below 
and select where you are. Use the companion to think through what you're 
noticing — then return to the page and write what you found.

Reclaiming your voice is not loud or dramatic. It's quiet, steady, and real. 
The companion is here to help you find the words.
```

4. Insert → Pictures → select `soul-quiet-qr.png`
5. Resize to **3cm square**
6. Center it
7. File → Save As → PDF
8. Upload to Etsy

**Option B: If you're using the PDF directly:**
1. Open The_Quiet_No_People_Pleasing.pdf in Adobe Acrobat Pro or similar
2. Go to the "How to Use" page
3. Insert the text (above)
4. Add the QR code image (3cm square, centered)
5. Save and re-upload

## Support Notes

### If Something Isn't Working

1. **QR code won't scan:** 
   - Make sure it's at least 2cm square
   - Check contrast (white background, black code)
   - Try scanning from different angles

2. **Companion won't load:**
   - Check that the URL is exactly correct
   - Verify you're accessing from a device with internet
   - Clear browser cache and try again

3. **Responses seem generic:**
   - The AI learns context from the section selected
   - Make sure you're selecting the right section first
   - The companion gets better with more interaction
