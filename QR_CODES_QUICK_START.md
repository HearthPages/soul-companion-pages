## WORKFLOW (Repeat 5x)

### 1. Generate QR Code (2 min per code)
1. Go to: **qr-code-generator.com**
2. Paste one URL above
3. Click Generate
4. Download as PNG
5. Name it: `soul-[name]-qr.png` (e.g., `soul-space-qr.png`)
6. **Scan it** with your phone to test it works

### 2. Add to Word Document (4 min per doc)
1. Open the corresponding `.docx` file
2. Find "How to Use" page
3. Go to end of page
4. Paste the companion text (see below)
5. Insert → Pictures → QR code PNG
6. Resize to **3cm square**
7. Center it
8. File → Save As → PDF (name: `..._Updated.pdf`)

### 3. Test & Verify (2 min per test)
1. Open the PDF
2. Scan QR code
3. Companion should load
4. Select a section
5. Verify prompt appears

## QUICK TROUBLESHOOTING

| Problem | Solution |
|---------|----------|
| QR won't scan | Make sure it's at least 2cm square (yours = 3cm ✓) |
| Companion won't load | Verify exact URL match, check internet |
| Wrong journal loads | Check URL journal parameter spelling |
| PDF quality poor | Make sure QR image is crisp in Word before saving |
