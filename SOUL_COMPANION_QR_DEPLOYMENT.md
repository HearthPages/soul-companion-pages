## STEP 3: Insert QR Code Images

### In Word (for each document):

1. **Position cursor** at the end of the companion text (after the last sentence)
2. Press **Enter** twice to create space
3. Go to **Insert → Pictures → This Device**
4. Select the corresponding QR code PNG file
5. **Resize** the image to approximately **3cm × 3cm** (1.2" × 1.2")
6. **Center** the image on the page
7. Optional: Add a caption below like *"Scan to access your Soul Companion"*

### Sizing Reference:
- **Too small:** QR code won't scan reliably
- **Too large:** Takes up too much page space
- **Just right:** 3cm square (about the size of a large postage stamp)

## Troubleshooting

### QR Code Won't Scan
- Make sure it's at least 2cm square (your 3cm is perfect)
- Check contrast — should be black on white background
- Try scanning from different angles
- Test with multiple phone cameras/QR scanner apps

### Companion Doesn't Load
- Check the URL is exactly correct (copy from table above)
- Verify you have internet access on the device
- Clear browser cache and try again
- Verify the journal parameter matches: `?journal=soul-space` (etc.)

### Sections Show Generic Prompts
- Verify you selected the correct section in the companion
- The AI learns context from what section you select
- Refresh the page and try again
- Make sure netlify deployment is active

### PDF Quality Issues
- In Word, ensure the QR code is crisp (not blurry)
- When saving to PDF, use "Standard" or "High" quality setting
- Test the PDF on screen first before printing
- If printing, use at least 300 DPI for best results
