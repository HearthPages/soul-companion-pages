## Instructions for each journal

1. Open the Word document
2. Go to the **How to Use This Journal** page
3. Add the text above at the bottom of that page
4. Go to **qr-code-generator.com**
5. Paste the URL for that journal and download the QR code as a PNG
6. In Word: **Insert → Pictures → This Device** — select the QR code
7. Resize to approx 3cm square and centre it
8. **File → Save As → PDF**
9. Re-upload the new PDF to Etsy

Repeat for all four journals.
