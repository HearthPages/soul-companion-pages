## Soul Companion File Naming

**Critical requirement:** Soul Companion deployments to Netlify must use `index.html` as the filename, not custom names like `netlify-deploy-FINAL.html` or `netlify-deploy-CORRECTED.html`.

- File location: `C:\Users\lisak\OneDrive\Documents\Claude\Projects\Hearth and Pages\Soul Companion\index.html`
- Deploy via drag-and-drop to Netlify dashboard at https://app.netlify.com
- Clear browser cache after deployment to see changes

## When Creating New Soul Companion Versions

Always save the final deployment file as **index.html** in the Soul Companion folder so it's ready to drag directly into Netlify.
